# 🧠 AI Proctoring System

An AI-based online exam monitoring system that ensures secure and fair examinations using real-time tracking, violation detection, and automated reporting.

---

## 🚀 Features

* 🎥 Real-time webcam monitoring
* 🧠 AI-based violation detection (tab switch, fullscreen exit, etc.)
* 📊 Admin dashboard with analytics
* 📝 Exam management system
* 📈 Leaderboard & results tracking
* 🔐 Secure authentication system

---

## 🛠️ Tech Stack

* **Frontend:** React.js, HTML, CSS, JavaScript
* **Backend:** Django (Python)
* **Database:** MySQL
* **AI/ML:** OpenCV
* **Cloud:** AWS

## ⚙️ Installation

```bash
# Clone repository
git clone https://github.com/your-username/AI-Proctor-System.git

# Go to backend
cd backend

# Install dependencies
pip install -r requirements.txt

# Run migrations
python manage.py migrate

# Start server
python manage.py runserver
```

---

## ▶️ Run Frontend

```bash
cd frontend
npm install
npm run dev
```

---

## 📌 Usage

1. Login as student/admin
2. Start exam
3. System monitors activity
4. Violations are recorded
